#  Grocery Store Management System (SQL Project)

##  Overview

This project analyzes a grocery store database to derive business insights related to sales, customers, suppliers, and employees.

##  Database Structure

* Customers
* Orders
* Order Details(Attributes-order_ID,quantity,product,priceEach,Total_price)
* Products
* Categories
* Suppliers
* Employees

## 📊 Key Analysis Performed

* Total revenue and order trends
* Top-selling products
* Supplier contribution to sales
* Employee performance analysis
* Monthly and weekday vs weekend trends

## 🛠️ Tools Used

* SQL (MySQL)

## 📈 Key Insights

* Identified peak sales months
* Found top-performing employees
* Analyzed customer purchase patterns

## 🚀 How to Run

1. Import the SQL file
2. Run table creation scripts
3. Execute analysis queries

## 📎 Project Files
* sql_create.sql(create db,table and insert, alter,update)
* proj_queries.sql(Analysis)
